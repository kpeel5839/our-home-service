# 가족 홈 서비스 (Family Home Service) 기획서

> 버전: v1.0
> 작성일: 2026-03-28
> 성격: 가족 전용 생활 관리 올인원 플랫폼

---

## 1. 프로젝트 개요

### 1.1 목적
가족 구성원들이 일상적인 집안일, 일정, 차량, 식사 등을 한 곳에서 편리하게 관리하고 공유할 수 있는 **가족 전용 스마트홈 관리 웹 서비스**

### 1.2 타겟 사용자
- 아파트에 거주하는 가족 단위 사용자 (2~6인 가구 기준)
- 주 사용 기기: 스마트폰 (모바일 퍼스트)
- 보조 기기: PC / 태블릿

### 1.3 핵심 가치
- **간편함**: 매일 반복되는 집안일을 클릭 몇 번으로 처리
- **연결**: 가족 구성원 간 실시간 공유 및 알림
- **자동화**: 카카오톡 알림으로 잊지 않게 챙겨주는 스마트 리마인더

## 4. 인증 및 가족 그룹

### 4.1 소셜 로그인
- **카카오 OAuth 2.0** 로그인
- 최초 로그인 시 닉네임, 프로필 사진, 역할(예: 엄마/아빠/아들/딸) 설정

### 4.2 가족 그룹
- 최초 가입자가 **가족 그룹 생성** → 초대 코드 발급
- 다른 가족 구성원은 초대 코드로 그룹 참여
- 그룹 내 구성원 관리 (이름, 역할, 프로필 이미지)
- 관리자 권한: 그룹 설정, 구성원 제거, 알림 규칙 관리

## 5. 주요 기능 명세

---

### 5.1 홈 대시보드

**화면 구성**
- 오늘 날짜 / 날씨 (선택)
- 오늘의 주요 정보 카드 모음
  - 오늘 쓰레기 담당자
  - 오늘의 메뉴
  - 귀가/외박 현황
  - 가까운 차량 예약
  - 유통기한 임박 식품 (3일 이내)
  - 오늘 청소 담당

---

### 5.2 쓰레기 담당 관리

**기능**
- 가족 구성원별 쓰레기 담당 일정 등록 (날짜 단위)
- 아파트 쓰레기 배출 가능 요일 설정 (예: 월/수/금)
- 배출 종류 분류: 일반쓰레기, 재활용, 음식물, 대형폐기물
- 담당자 완료 체크 기능

**알림**
- 배출 가능 전날 저녁 / 당일 아침 → 담당자에게 카카오톡 알림
- 예시: "내일은 재활용 배출일이에요. 담당: 홍길동"

**데이터 모델**
```
TrashSchedule {
  id, date, memberId, trashType, isCompleted, createdAt
}
ApartmentTrashRule {
  id, dayOfWeek (0-6), trashTypes[], familyGroupId
}
```

---

### 5.3 오늘의 메뉴

**기능**
- 아침/점심/저녁 메뉴 등록 (텍스트 + 선택적 사진)
- 가족 구성원 누구나 등록 및 수정 가능
- 작성자 표시
- 주간 메뉴 히스토리 조회

**알림**
- 매일 정해진 시간 (예: 오전 9시)에 오늘 메뉴 미등록 시 → 등록 독려 알림
- 메뉴 등록 시 → 가족 전체에게 알림 (선택 설정)

**데이터 모델**
```
DailyMenu {
  id, date, mealType (BREAKFAST|LUNCH|DINNER),
  menuName, description, imageUrl,
  registeredBy (memberId), createdAt
}
```

---

### 5.4 귀가 / 외박 현황

**기능**
- 가족 구성원별 오늘 상태 등록
  - 귀가 예정 (예상 시간 선택 가능)
  - 외박
  - 미정
- 상태 변경 가능 (당일 중 변경 허용)
- 홈 대시보드에서 가족 전체 현황 한눈에 확인

**알림**
- 상태 변경 시 가족 전체 알림 (선택 설정)
- 예시: "홍길동님이 오늘 외박 예정이에요."

**데이터 모델**
```
AttendanceStatus {
  id, date, memberId,
  status (HOME|OUT|UNKNOWN),
  expectedReturnTime, memo, updatedAt
}
```

---

### 5.5 차량 관리 시스템

#### 5.5.1 차량 예약

**기능**
- 등록된 차량 목록 (가족 차량 1~N대 등록 가능)
- 날짜 + 시간대 기준으로 차량 사용 예약
- 예약 시 목적 메모 입력 가능
- 예약 충돌 방지 로직 (겹치는 시간대 예약 불가)
- 본인 예약 취소 가능

#### 5.5.2 주유 기록

**기능**
- 주유 날짜, 주유량(L), 금액, 주유소명 기록
- 누적 주유 이력 조회
- 평균 연비 계산 (선택 기능)

#### 5.5.3 주차 위치 기록

**기능**
- 주차 완료 후 위치 기록: 층수, 구역, 메모
- 사진 첨부 가능 (주차 위치 사진)
- 현재 차량이 어디에 있는지 가족 전체에서 조회 가능

**알림 (핵심 기능)**
- 차량 사용 예약자에게 **당일 저녁 (예: 오후 8시)** 자동 카카오톡 알림
- 알림 내용: "오늘 [차량명] 사용하셨나요? 주차 위치를 기록해주세요!"
- 주차 위치가 이미 입력된 경우: "오늘 [차량명]은 [B2층 A-12구역]에 주차되어 있어요."

**데이터 모델**
```
Vehicle {
  id, name, plateNumber, familyGroupId
}
VehicleReservation {
  id, vehicleId, memberId, startTime, endTime, purpose, createdAt
}
FuelRecord {
  id, vehicleId, memberId, date, liters, amount, stationName
}
ParkingRecord {
  id, vehicleId, memberId, floor, zone, memo, imageUrl, recordedAt
}
```

---

### 5.6 커뮤니티 (가족 피드)

**기능**
- 사진 + 텍스트 게시글 작성
- 이미지 다중 업로드 (최대 10장)
- 가족 구성원만 조회 가능 (비공개 피드)
- 좋아요 / 댓글 기능
- 최신순 / 날짜별 조회
- 월별 앨범 뷰 (사진만 모아보기)

**데이터 모델**
```
Post {
  id, authorId, content, images[], createdAt, updatedAt
}
Comment {
  id, postId, authorId, content, createdAt
}
Like {
  id, postId, memberId, createdAt
}
```

---

### 5.7 청소 담당 관리

**기능**
- 청소 항목 등록 (예: 화장실 청소, 거실 청소, 설거지, 방 청소 등)
- 각 항목별 담당자 + 주기 설정 (매일 / 매주 특정 요일 / 격주 / 매월)
- 담당자 완료 체크
- 담당 순번 자동 로테이션 설정 가능

**알림**
- 해당 날짜 아침 (예: 오전 8시) → 담당자에게 카카오톡 알림
- 예시: "오늘은 [화장실 청소] 담당이에요! (담당: 홍길동)"
- 미완료 시 저녁 리마인더 (선택 설정)

**데이터 모델**
```
CleaningTask {
  id, name, familyGroupId, frequency (DAILY|WEEKLY|BIWEEKLY|MONTHLY),
  dayOfWeek, assignmentType (FIXED|ROTATION)
}
CleaningAssignment {
  id, taskId, memberId, scheduledDate, isCompleted, completedAt
}
```

---

### 5.8 냉장고 재고 관리

**기능**
- 식품 등록: 이름, 카테고리, 수량, 유통기한, 보관 위치 (냉장/냉동/실온)
- 최근 등록 순으로 목록 표시
- 유통기한 D-day 표시 (임박 시 빨간색 강조)
- 소비 완료 처리 (완료 시 목록에서 제거 또는 숨김)
- 카테고리 필터 (채소, 육류, 음료, 유제품, 반찬 등)
- 검색 기능

**알림**
- 유통기한 3일 전 → 가족 전체 카카오톡 알림
- 예시: "냉장고 속 [삼겹살]의 유통기한이 3일 후예요! 빨리 드세요~"

**데이터 모델**
```
FridgeItem {
  id, familyGroupId, registeredBy, name, category,
  quantity, unit, expirationDate, storageType,
  isConsumed, createdAt, updatedAt
}
```

## 6. 카카오톡 알림 정책

| 알림 유형 | 발송 시점 | 수신 대상 | 기본값 |
|----------|----------|---------|--------|
| 쓰레기 배출 알림 | 배출 전날 오후 8시 / 당일 오전 7시 | 담당 구성원 | **ON** |
| 청소 담당 알림 | 당일 오전 8시 | 담당 구성원 | **ON** |
| 청소 미완료 리마인더 | 당일 오후 7시 (미완료 시) | 담당 구성원 | OFF |
| 차량 주차 위치 알림 | 당일 오후 8시 | 차량 사용 예약자 | **ON** |
| 유통기한 임박 알림 | 유통기한 3일 전 오전 9시 | 가족 전체 | **ON** |
| 귀가/외박 상태 변경 | 상태 변경 즉시 | 가족 전체 | OFF |
| 메뉴 등록 알림 | 등록 즉시 | 가족 전체 | OFF |

> 알림은 카카오 알림톡 API 또는 카카오 비즈메시지를 통해 서버에서 발송
> 각 알림은 설정에서 구성원별로 개별 ON/OFF 가능
> 기본값 OFF 항목은 원하는 경우 설정에서 직접 켤 수 있음

## 11. 디자인 방향

- **톤앤매너**: 따뜻하고 친근한 가족 감성 (파스텔 계열 또는 화이트 + 포인트 컬러)
- **컴포넌트**: 카드 기반 UI, 충분한 여백
- **아이콘**: 직관적인 아이콘 (Heroicons 또는 Lucide)
- **폰트**: 한국어 가독성 높은 폰트 (Noto Sans KR 또는 Pretendard)
- **인터랙션**: 부드러운 트랜지션, 햅틱 피드백 지원 (모바일)
